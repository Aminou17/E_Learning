Découvrir Linux

Linux est un système d’exploitation très utilisé par les développeurs et les serveurs web. Il permet de gérer un ordinateur et d’exécuter des programmes à l’aide de commandes dans le terminal.

Dans cette vidéo vous allez découvrir les bases de Linux et apprendre les commandes essentielles pour naviguer dans les dossiers, créer des fichiers et mieux comprendre l’environnement utilisé par les développeurs.
