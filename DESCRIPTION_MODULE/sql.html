Découvrir SQL

SQL est un langage utilisé pour travailler avec les bases de données. Une base de données sert à stocker des informations comme des comptes utilisateurs, des produits ou des messages dans un site web.

Dans cette vidéo vous allez découvrir les bases de SQL et comprendre comment lire, ajouter, modifier ou supprimer des données dans une base de données grâce à des commandes simples.
