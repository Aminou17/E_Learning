Découvrir PHP

PHP est un langage de programmation utilisé pour créer des sites web dynamiques. Contrairement au HTML, il permet de traiter des informations sur le serveur avant d'envoyer la page au navigateur.

Dans cette vidéo vous allez découvrir les bases de PHP et comprendre comment récupérer des données d’un utilisateur, traiter des formulaires et interagir avec une base de données pour rendre un site web plus interactif.
